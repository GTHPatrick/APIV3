API Tiny ERP - Categorias

GET /categorias/todas

Descrição: Retorna toda a árvore de categorias cadastradas no sistema.

Parâmetros: Nenhum

Resposta 200 - OK

Content-Type: application/json

Exemplo de Resposta:

{
  "id": 0,
  "descricao": "string",
  "filhas": [
    "string"
  ]
}

Resposta 400 - Bad Request

Content-Type: application/json

Exemplo de Resposta:

{
  "mensagem": "string",
  "detalhes": [
    "string"
  ]
}
